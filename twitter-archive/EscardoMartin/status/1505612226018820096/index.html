Let's identify the set P with a proposition. I know this:
(i) If P holds, or if ¬P holds, then P->2 is discrete.
(ii) If P->2 is discrete, then ¬P holds or ¬¬P holds.

Can we reverse the implication (i) or the implication (ii)?
Certainly we can't reverse both, because this would mean that De Morgan's Law implies Excluded middle, which is known to fail in some models.
