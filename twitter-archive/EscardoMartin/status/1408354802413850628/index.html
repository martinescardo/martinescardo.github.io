Say that a type X is *compact* if for every decidable property p of elements of X, the statement "there is x:X with p(x)" is decidable. That is, either there is x:X with p(x) or else p(x) is false for all x:X.

Concretely, we can construct ℕ∞ as the type of infinite binary sequences α such that αₙ ≥ αₙ₊₁ for all n. The sequence of n-many 1's followed by infinitely many 0's is a copy of the natural number n:ℕ in ℕ∞. The sequence of infinitely many 1's is the new point ∞.

We define the type "squashed-sumₙ Xₙ" to be
Σ(i:ℕ∞), Π(n:ℕ), (i=n → Xₙ)
so that if each Xₙ is compact then the type squashed-sumₙ Xₙ is also compact.
