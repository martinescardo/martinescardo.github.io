In classical mathematics, there is no difference between "∃x,A(x)" and "¬∀x,¬A(x)", where A is a mathematical statement about the mathematical object x, and "∃", "∀" and "¬" stands for "exists", "for all" and "not" .

In the internal language of a topos, we have the following rather useful but highly misleading "existence property", which is a meta-theorem:

If we can prove "∃x,A(x)" in an empty context, then we can find a term t in the internal language and a proof of A(t).
