In more detail, fix a universe 𝓤, and consider the sum operator Σ: (Y:𝓤) → (Y → 𝓤) → 𝓤. Then for any Y:𝓤, we have that the fiber of the map Σ Y : (Y → 𝓤) → 𝓤 at X:𝓤 is equivalent to the function type (X → Y).
