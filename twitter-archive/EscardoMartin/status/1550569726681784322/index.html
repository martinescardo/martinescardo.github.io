An example I like is MLTT withoout any universe. Then it is known that you can only define (codes of) ordinals < ε₀. But if you add just one universe, you can define ε₀ (and more). See e.g. this in Agda (which is not original work): https://www.cs.bham.ac.uk/~mhe/TypeTopology/Ordinals.Codes.html
