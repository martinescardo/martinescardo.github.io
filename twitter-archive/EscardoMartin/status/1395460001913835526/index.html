@XenaProject @jonmsterling Let ℕ∞ be the type of maps α:ℕ→2 s.t. α(n)=0 ⇒ α(n+1)=0. Define a point ∞:ℕ∞ by ∞(n)=0. Define ι:ℕ+1→ℕ∞ by sending k:ℕ to α defined by α(n)=1 ⇔ n<k, and the new point to ∞. This map ι is an injection, and classically, but not constructively, a surjection.
