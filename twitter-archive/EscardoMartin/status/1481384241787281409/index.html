Martin Escardo

@EscardoMartin

Consider the following problem:

(A : ℝ → Ω)
→ ((Σ x ꞉ ℝ , x ∈ A) ≃ ℕ∞)
→ (x : ℝ) → x ∈ A → locator x

This says that if A is a subset of ℝ isomorphic to ℕ∞, then every element of A is equipped with a locator.
