Theorem. There is a program that prints "yes" if it's going to rain tomorrow and "no" otherwise.
Proof. If it's going to rain tomorrow, we can take the program to be "print yes". Otherwise we can take it to be "print no". Q.E.D.

The fact that every computable number has a computable binary expansion is proved very much in the same way. If the given number is rational, then use the obvious program. Otherwise, use an alternative program. But we can't tell, computably, whether a given number is rational.
