Start with a binary relation '<' on a set X. Then say, by induction, that an element x of X is accessible if all its predecessors are accessible. (The (vacuous) base case is when there are no predecessors.)

Martin Escardo

@EscardoMartin

A well-ordered set is such an accessible pair (X,<) additionally satisfying
(1) Any two elements with the same predecessors are equal.
(2) "<" is transitive.
