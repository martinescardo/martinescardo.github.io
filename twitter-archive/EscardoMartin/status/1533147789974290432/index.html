Martin Escardo

@EscardoMartin

The concept is this, named "green" for the purposes of this thread.

A set X is green if for any function f : X -> {0,1} we can either exhibit a root of p or tell there is none.

(I don't want to commit to any foundation here, set theory or type theory or whatever.)
