In the presence of excluded middle, there is an equivalent, inductive way of saying the same thing, which is more suited for constructive mathematics.

Given a relation "<" on a set X, say that an element x:X is accessible, with respect to that relation, if every y:X with y<x is accessible. (The base case of the inductive definition occurs when there are no y:X with y<x.)
