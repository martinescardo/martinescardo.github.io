Prove the following instance of the principle of excluded middle:
For any given function f: (ℕ → 2) → ℕ, we have that either f(δ(n)) = 0 all n ∈ ℕ, or not.
(We are not assuming that f is continuous. Notice that the universal quantification is over ℕ.)
