Martin Escardo

People want canonical representatives of well-ordered sets. In ZFC set theory, such a canonical choice is given by the so-called von neumann ordinals.

Every set equipped with a well-order is uniquely order-isomorphic to a von neumann ordinal.

Martin Escardo

@EscardoMartin

An ordinal is a type α equipped with a relation _<_ satisfying the following conditions:
(i) _<_ is transitive.
(ii) If two elements have the same predecessors then they are equal.
(iii) _<_ satisfies the principle of transfinite induction.

Martin Escardo

The principle of transfinite induction is the following:

For any property P of elements of X, if we have

∀ x : X , (∀ y : X , y < x → P y) → P x

then

∀ x : X , P x
