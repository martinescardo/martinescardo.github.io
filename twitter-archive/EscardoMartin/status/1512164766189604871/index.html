Dubuc and Penon proposed the following beautiful definition of the compactness of a set A (as opposed to the compactness of a topological space A):
∀A∈Ω ∀B∈(X→Ω)(∀x∈X(A∨B(x)) ⇒ A∨∀x∈X(B(x))).
Here Ω is the object of truth values in an arbitrary topos.
