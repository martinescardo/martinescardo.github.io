If x is a natural number, (x+1)! = (x+1) × x!
If X is a type and X! is the type of automorphisms of X, and X has decidable equality, (X+1)! ≃ (X+1) × X!
In general, (X+1)! ≃ (X+1)' × X! where (X+1)' is the coderived set of the type X+1. Puzzle: what does coderived set mean?
