@LawrPaulson @freekwiedijk @EgbertRijke @XenaProject @andrejbauer @SidGadgil Can we embed the type of 0-groupoids into a 1-groupoid? No.
But then we have to understand *embedding* in the correct sense of UF.
A function f:X->Y is an embedding iff for every y:Y the type Sigma x:X, Id Y (f x) y is a subsingleton (i.e. contractible if pointed).
