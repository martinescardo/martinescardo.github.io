@wtgowers @RWusthofen Let 0 be the empty set and 1 be some chosen set with precisely one element. The above truth table corresponds, precisely, to the following facts, if we identify "false" with the empty set 0 and "true" with the one-element set 1.

0 ⊆ 0
0 ⊆ 1
1 ⊊ 0
1 ⊆ 1
